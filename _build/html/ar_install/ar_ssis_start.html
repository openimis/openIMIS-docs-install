Execute SSIS¶

Once both the SSIS and SSAS packages are deployed successfully, it’s time to start the ETL process. To start the ETL Process, follow the instructions below.

  1. Navigate to the folder where the SSIS package was installed. Find the file named “IMISDW” and double click the package to begin the process(Image - Run SSIS).

    Image - Run SSIS

  2. This process might take a while to finish depending on the data volume. Once the process is completed successfully, the SSAS package is now ready for the reporting(Image - SSIS Results).

    Image - SSIS Results
